Dummy API คืออะไร?

Dummy API คือ API สำหรับสร้างชุดข้อมูลตัวอย่าง เช่น ข้อมูลสินค้า ข้อมูลบุคคล เป็นต้น
API ตัวนี้ออกแบบมาเพื่อให้น้องๆเพื่อนๆพี่ๆที่เริ่มใช้ API ได้ลองเล่นและใช้งาน เพื่อนทำความเข้าใจและศึกษาการทำงานของ RESTful API และยังมี Source Code ที่ผมได้ทำเอาใว้ สำหรับท่านใดที่อยากศึกษาสามารถนำโค้ตผมไปใช้ได้เลย แต่ไม่อนุญาตให้นำโค้ตชุดนี้ไปขายหรือกระทำการใดๆในทางที่ไม่ดี

วิธีการใช้งาน

ผมไม่ได้ทำคำแนะนำใว้เป็นเว็บเพจนี้ เนื่องจากใช้เวลานานและออกแบบยากเกินไป ผมเลยเขียนทุกอย่างใว้บน Github แทน ผมได้เขียนรายละเอียดและวิธีการใช้งานใว้ที่ Github ของผมแล้ว ทุกๆท่านสามารถเข้าไปอ่านได้เลยครับผม

* หากใครมี Feedback หรืออนาคต API v2 อยากให้มีระบบอะไรเพิ่มขึ้นมา(ที่เป็นไปได้) ผมจะทำมาเป็นตัวอย่างให้ครับผม
